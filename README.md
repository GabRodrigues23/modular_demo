### O que é o Flutter Modular?

O `flutter_modular` é um pacote para o ecossistema Flutter que ajuda a estruturar e organizar o código do projeto de forma escalável. Seus principais objetivos são:

* **Modularização:** Permitir que o projeto seja dividido em módulos independentes, facilitando a manutenção e o reuso de código.
* **Injeção de Dependência:** Fornecer um sistema inteligente para gerenciar e injetar dependências (como controllers, repositories, etc.) de forma automática.
* **Gerenciamento de Rotas:** Criar um sistema de rotas nomeadas e hierárquicas, simplificando a navegação entre as telas da aplicação.
